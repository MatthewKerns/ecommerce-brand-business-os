## MADA's Strategy (What They Do For $2K/mo)

### The Affiliate Funnel Model

```
Optimize Shop → Seed Reviews → Mass Affiliate Outreach → Provide Scripts →
Receive Affiliate Videos → Run Ads on Best Videos → Scale
```

### Step-by-Step Breakdown

1. **Listing Optimization**: Main images, keywords, descriptions, video
2. **Review Seeding**: 20 initial purchases with prepared review bank (specific keywords/sentences)
3. **Daily Affiliate Outreach**: Target sellers with >$1K/mo GMV in last 30 days
4. **Script Distribution**: Analyze top competitors, prepare branded scripts
5. **Free Sample Fulfillment**: Ship product + printed script in one package
6. **Follow-Up Cadence**: 3x/week follow-ups asking affiliates to post videos using scripts
7. **Ad Campaign Management**: Run ads on top-performing affiliate videos (their page, our ad spend)
8. **Full Shop Management**: Customer messages, orders, shop health scores, violations, campaigns

### Key Insight: Product Listing = Landing Page
> "For us, product listing is like a website landing page. We optimize it well before driving any traffic to it. Then we drive traffic through affiliate videos and lives."

### The Funnel Metrics They Track
- Views → Clicks → Conversion Rate
- Optimize by changing: scripts, main image, carousel images, flash sales, coupons, discounts

## TikTok Shop Affiliate Outreach Tiers (CRITICAL)

### Tier System for Seller Outreach Limits

| Tier | Requirement | Weekly Outreach Limit | Audience |
|------|------------|----------------------|----------|
| **Tier 1** | New shop (no affiliate sales) | 1,000 actions (one-time) | Limited pool |
| **Tier 2** | 1st affiliate sale ($1 minimum) | 2,000/week | Micro-influencers (<20K followers) |
| **Tier 3** | $2,000 affiliate GMV in 30 days | 7,000/week | **ANY affiliate (no follower limit)** |

### Unlocking Tier 2 (Immediate Action Required)
- Make **one self-purchase through an affiliate link** to unlock 2,000 weekly outreach actions
- Resets on Sundays
- Only need $1 in affiliate GMV to unlock

### Unlocking Tier 3 (The Real Goal)
- Hit $2,000 in affiliate GMV within 30 days
- Unlocks 7,000 weekly outreach actions with **no follower limits**
- This is the key scaling unlock - 70% of TikTok sellers never reach this tier
- Without this, you're competing with hundreds of thousands of sellers for the same micro-influencer pool

### Why Tier 3 Matters
> "70% of sellers are reaching out to the same small pool of micro-influencers under 20K followers. This is the most competitive market on TikTok. They receive hundreds of offers each day."

## What We're Building Instead (DIY Playbook)

### Immediate Actions
1. **Self-purchase through affiliate link** → Unlock Tier 2 (2,000 weekly outreach)
2. **Optimize TikTok Shop listings** → Treat as landing pages
3. **Seed 10-20 reviews** → Use prepared review bank with strategic keywords
4. **Build affiliate outreach automation** → Replicate MADA's daily mass outreach via API

### Technical Implementation (Organic Marketing Package)
1. **TikTok Shop Affiliate API Integration** → Search creators, filter by GMV, send invitations
2. **Automated Script Generation** → Analyze top competitors, generate video scripts
3. **Outreach Campaign Manager** → Daily batch outreach with templates + follow-up cadence
4. **Performance Tracking** → Track affiliate GMV, video views, conversion rates
5. **Script Scraping** → Use tools like Product Hunter AI to analyze high-performing videos

### Scale Path
```
Week 1: Self-purchase → Unlock Tier 2 (2K outreach/week)
Week 2-4: Mass outreach to micro-influencers, send samples + scripts
Week 4-8: Hit $2K affiliate GMV → Unlock Tier 3 (7K outreach/week, no limits)
Week 8+: Scale with unlimited outreach to top affiliates
```

### Cost Comparison
- **MADA**: $2,000/month + ad spend
- **DIY**: Free sample cost (~$10-15/unit) + shipping + our time (automated)
- **Savings**: $2,000/month = $24,000/year
