## Values in Action

### Decision Framework
When facing a decision, ask:
1. Does this align with our vision?
2. Does this serve our mission?
3. Does this honor our values?

If NO to any: **Don't do it.**

### Daily Application
- **Morning**: Review one value to focus on
- **Decisions**: Run through values checklist
- **Evening**: Assess values alignment for the day
