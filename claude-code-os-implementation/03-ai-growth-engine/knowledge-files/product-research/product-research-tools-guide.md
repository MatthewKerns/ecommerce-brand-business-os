# Product Research Tools Guide

> A comprehensive guide to tools for each phase of product research - from idea discovery to IP protection.

---

## Overview: The Five Phases

| Phase | Focus | Key Question |
|-------|-------|--------------|
| 1. Finding Inspiration | Trend spotting, design inspiration | "What's gaining momentum?" |
| 2. Competitive Analysis | Market understanding, social listening | "What exists and what's missing?" |
| 3. Deep Dives | Quantitative data, platform-specific research | "What do the numbers say?" |
| 4. Pipeline Management | Organization, task management | "How do we stay organized?" |
| 5. IP Protection | Patents, trademarks, legal protection | "How do we protect our ideas?" |

### Pipeline Structure Recommendation

```
PRODUCT PIPELINE
├── Phase 1: Ideas (Raw concepts)
├── Phase 2: Research (Competitive analysis complete)
├── Phase 3: Validation (Data gathered, opportunity scored)
├── Phase 4: Development (Supplier sourcing, samples)
├── Phase 5: Launch Prep (Listing, imagery, inventory)
└── Archive (Rejected ideas with notes)
```

## Phase 5: Protecting Your Intellectual Property (IP)

### IP Resources

| Resource | URL | Function |
|----------|-----|----------|
| ACID (UK) | acid.uk.com | IP advice and membership |
| Zero to IP | zerotoip.com | Rey Barcelo's IP AI Tool |

### IP Considerations

| Type | Protects | Timeline |
|------|----------|----------|
| **Trademark** | Brand name, logo | Register before launch |
| **Design Patent** | Unique appearance | File early in development |
| **Utility Patent** | Function/mechanism | Requires significant investment |
| **Copyright** | Creative works, images | Automatic but register for enforcement |
