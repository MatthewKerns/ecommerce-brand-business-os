# Blog

Next.js blog for infinitycards.com with SEO and AEO optimization.

## Features

- SEO-optimized content rendering
- Structured data (JSON-LD) for search engines
- AEO-optimized content for AI assistant citations
- Content management with markdown posts
- Vercel deployment configuration

## Setup

```bash
cd packages/blog
npm install
```

## Development

```bash
npm run dev    # Start development server
npm run build  # Production build
```
