# Fiverr Hiring SOP

*Standard Operating Procedure for finding and hiring freelance designers*

---

## When to Use Fiverr

| Use Case | Fiverr | Alternative |
|----------|--------|-------------|
| One-off main image | ✅ Yes | - |
| Quick A/B test variant | ✅ Yes | - |
| Full listing image set | ✅ Yes | DesignForce if subscribed |
| Ongoing monthly work | ⚠️ Maybe | Consider DesignForce |
| Complex A+ content | ✅ Yes | Agency if budget allows |
| Rush job (24-48hr) | ⚠️ Limited | Direct designer contact |

---

## Step 1: Search for Designers

### Search Terms to Use

**Main Images / Product Photography:**
- "Amazon product photography"
- "Amazon main image design"
- "3D product render"
- "Product mockup design"
- "Amazon listing images"

**Infographics / Secondary Images:**
- "Amazon infographic design"
- "Amazon listing infographic"
- "Product infographic"
- "Amazon secondary images"

**A+ Content:**
- "Amazon A+ content design"
- "Amazon EBC design"
- "Amazon enhanced brand content"

### Search Filters

| Filter | Setting | Why |
|--------|---------|-----|
| Seller Level | Level 2 or Top Rated | Quality assurance |
| Delivery Time | Match your deadline | Realistic expectations |
| Price | Your budget range | Stay within limits |
| Reviews | 4.8+ stars | Consistent quality |

## Step 2: Evaluate Sellers

### Quick Scan Checklist (30 seconds each)

- [ ] Portfolio has Amazon product images
- [ ] Style matches "premium" feel
- [ ] Reviews mention good communication
- [ ] Delivery time works for deadline
- [ ] Price within budget

### Deep Evaluation (top 3-5 candidates)

**Portfolio Review:**
| Question | Good Sign | Red Flag |
|----------|-----------|----------|
| Style match? | Premium, clean, professional | Cartoonish, budget-looking |
| Amazon experience? | Clear Amazon listings in portfolio | Only social media graphics |
| Variety? | Multiple product types | Only one style |
| Quality consistency? | All work looks good | Mixed quality |

**Reviews Analysis:**
| Look For | Avoid |
|----------|-------|
| "Great communication" | "Hard to reach" |
| "Understood my vision" | "Had to explain multiple times" |
| "Quick revisions" | "Slow to respond" |
| "Exceeded expectations" | "Just okay" |

**Gig Details Check:**
- What's included in base price?
- How many revisions?
- Source files included?
- What extras cost more?

## Step 3: Send Inquiry Message

### Pre-Order Message Template

```
Hi [Name],

I'm looking for [ASSET TYPE] for my Amazon listing.

BRAND: Infinity Vault - Premium trading card storage (card binders, deck boxes)
STYLE: Premium, battle-ready aesthetic, dramatic lighting, black + metallic gold/silver

WHAT I NEED:
- [Deliverable 1]
- [Deliverable 2]
- [Deliverable 3]

I have ready to share:
✓ Product photos/3D files
✓ Brand guidelines with colors/fonts
✓ Reference images for style direction
✓ Technical specifications

TIMELINE: Need first draft by [DATE]

Questions:
1. Can you match this premium style? [attach reference]
2. What's your actual turnaround for first draft?
3. How many revisions included?
4. Source file included?

Thanks!
[Your name]
```

### What to Attach:
- 1-2 reference images showing desired style
- Current product photo (if applicable)
- Competitor example (if differentiation needed)

### Evaluate Responses:
| Good Response | Bad Response |
|---------------|--------------|
| Answers all questions | Ignores questions |
| Asks clarifying questions | Generic reply |
| Shows enthusiasm | Copy-paste feel |
| Provides realistic timeline | Overpromises |

```
1. [Specific requirement 1]
2. [Specific requirement 2]

- Size: [dimensions]
- Format: [file types needed]
- Background: [white/transparent/other]

```

### Attachments Checklist:
- [ ] Design brief (or key excerpts)
- [ ] Product images/3D files
- [ ] Warranty badge asset
- [ ] Brand color codes
- [ ] Reference images (2-3)
- [ ] Current image (if redesign)

## Step 5: Review & Feedback

### First Draft Review (within 24 hours)

**Quick Assessment:**
- [ ] Matches requested style?
- [ ] Brand colors correct?
- [ ] Technical specs met?
- [ ] Key elements included?

**Gut Check:**
- Would I click this?
- Does it look premium?
- Does it stand out?

### Feedback Format

```
Hi [Name],

Thanks for the first draft!

OVERALL: [Good start / Needs some work / Needs major revision]

WHAT'S WORKING:
• [Positive element 1]
• [Positive element 2]

CHANGES NEEDED:
• [Change 1] - [Specific instruction]
• [Change 2] - [Specific instruction]
• [Change 3] - [Specific instruction]

PRIORITY: [Change X] is most important.

Questions: [Any clarifications needed]

Thanks!
```

### Revision Rules:
- Be specific, not vague ("move badge 20px left" not "badge looks off")
- Reference visual examples when possible
- Prioritize changes (what matters most?)
- Stay within revision limits or expect extra cost

---

## Step 6: Final Delivery

### Acceptance Checklist

- [ ] All requested files delivered
- [ ] Correct dimensions/formats
- [ ] Source files included (if ordered)
- [ ] Layers organized (if PSD/AI)
- [ ] Final matches approved revision

### Post-Delivery Actions

1. **Download all files** (Fiverr removes after time)
2. **Organize in folder**: `designs/delivered/[ASIN]-[Date]/`
3. **Rate seller** (helps future searches)
4. **Add to vendor list** (if good experience)

## Budget Guidelines

### Price Ranges (USD)

| Asset | Budget | Mid-Range | Premium |
|-------|--------|-----------|---------|
| Single main image | $30-50 | $50-100 | $100-200 |
| Image set (6-7) | $100-150 | $150-300 | $300-500 |
| A+ content (full) | $150-250 | $250-400 | $400-700 |
| Infographic (single) | $25-40 | $40-75 | $75-150 |
| 3D render | $50-100 | $100-200 | $200-400 |

### When to Pay More:
- Complex product requiring 3D work
- Rush timeline
- Very specific style requirements
- Need for multiple variations
- Source files required

### When Budget is Fine:
- Simple product photography
- Standard infographic layout
- Following existing template
- Flexible timeline

## Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Seller unresponsive | Message after 24hrs, then escalate |
| Work doesn't match portfolio | Ask for revision citing portfolio example |
| Missing deadline | Contact before deadline, request update |
| Quality issues | Specific feedback, reference original requirements |
| Scope creep charges | Clarify in original order, push back if unfair |

### When to Cancel:
- Seller completely unresponsive (48+ hours)
- Work is drastically different from portfolio
- Seller refuses reasonable revisions
- Major deadline miss with no communication

### How to Cancel:
1. Message seller explaining issue
2. Request resolution or mutual cancellation
3. If no response, use Fiverr resolution center
4. Document everything for dispute
