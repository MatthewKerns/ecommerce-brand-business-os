## SECTION 2: BRAND REQUIREMENTS

### Brand Identity (NON-NEGOTIABLE)
**Essence**: Battle-Ready Empowerment
**Tagline**: "Show Up Battle Ready"
**Archetypes**: The Hero + The Magician

### Visual Requirements

#### Color Palette
| Color | Hex Code | Usage |
|-------|----------|-------|
| Primary Black | #0D0D0D | Background, authority |
| Metallic Gold | #D4AF37 | Accents, premium signals |
| Metallic Silver | #C0C0C0 | Secondary accents |
| Deep Purple | #2D1B4E | Fantasy accents (optional) |

#### Typography Direction
- Bold, clean, authoritative
- Modern with subtle fantasy influences
- No script fonts, no cartoonish fonts

#### Imagery Style
- High-contrast, dramatic lighting
- Product as hero (center focus)
- Premium, aspirational feel
- Fantasy-influenced but NOT cartoonish
- Gaming context when appropriate

### Language/Text on Images
**Words We Use**:
- Battle-ready, equipment, gear, arsenal
- Premium, professional-grade
- Lifetime warranty

**Words We NEVER Use**:
- Storage, container, box
- Cheap, budget, basic
- Simple, standard, ordinary

## SECTION 3: CTR/CVR OPTIMIZATION REQUIREMENTS

### For Main Images (CTR Focus)
The goal is to WIN THE CLICK in search results.

**Required Elements**:
1. [ ] Lifetime Warranty badge (visible, prominent)
2. [ ] Product clearly visible and hero-positioned
3. [ ] Premium quality signals (material texture, lighting)
4. [ ] Differentiation from competitors (unique angle/composition)
5. [ ] White background OR context that pops on Amazon grid

**CTR Testing Principles**:
- Single variable testing (change ONE thing per test)
- 7-day minimum test period
- Test against current winner, not multiple variants

**Competitor Differentiation**:
| Competitor Style | Our Differentiation |
|------------------|---------------------|
| Generic product shot | Dramatic lighting, premium feel |
| No trust signals | Lifetime warranty badge visible |
| Commodity appearance | Battle-ready aesthetic |
| Cluttered images | Clean, focused composition |

### For Secondary/A+ Content (CVR Focus)
The goal is to CONVERT the click into a sale.

**Required Elements**:
1. [ ] Emotional benefit messaging (identity, not utility)
2. [ ] Social proof indicators
3. [ ] Quality/material callouts
4. [ ] Use case scenarios
5. [ ] Brand story connection
